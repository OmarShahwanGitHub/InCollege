======================================
MAIN-MENU -> *REGISTRATION (main menu: 2. Create New Account)

IF count >= 5
DISPLAY "All permitted accounts have been created, Max 5 accounts."
EXIT PARAGRAPH
END-IF

DISPLAY "Enter Username:"
DISPLAY "Enter Password:"
WHILE VALID-PASS = FALSE
VALIDATE IF INVALID WILL DISPLAY "Error: Password must be 8-12 characters long."
END-WHILE
IF VALID
DISPLAY "Account successfully created!" -> *MAIN-MENU
======================================


======================================
MAIN-MENU -> *LOGIN-PROCESS (1. Log in)

DISPLAY "Please enter your username:"
DISPLAY "Please enter your password:"
VALIDATE
IF VALID
DISPLAY "WELCOME" -> *POST-LOGIN-MENU
ELSE
   DISPLAY "Incorrect username/password, try again."
END-IF
